TANNA – Stufenstruktur OP-0 bis OP-7
====================================

Projektübersicht:
-----------------
Die hier enthaltenen PNG-Dateien zeigen die geometrisch und visuell verfeinerte Entwicklungsstufe des TANNA-Symbols
von OP-0 bis OP-7. Jede Stufe repräsentiert eine abstrakte Schichtung oder Wachstumsebene innerhalb eines logischen,
grafischen oder systemischen Baumschemas.

-------------
- tanna_op0_centered.png     : Gleichschenkliges Dreieck, Basisstufe OP-0 (dunkelblau, zentriert)
- tanna_op1_fixed.png        : OP-0 mit Stamm (1/3 Breite, 1/12 Höhe), Farbe leicht grünlicher
- tanna_op2_visible_clear.png: OP-1 + zweites Dreieck (1/2 Höhe), klar erkennbar gestaffelt
- tanna_op3_refined_sharp.png: OP-2 + drittes Dreieck, schärfer verjüngt
- tanna_op4_refined.png      : Vierfach gestaffelte Tanne mit klarer, harmonischer Verjüngung
- tanna_op5_op7like.png      : Fünffach gestaffelt, visuell abgestimmt auf spätere OP-7
- tanna_op6_refined.png      : Sechsstufige Tanne mit durchgehender Proportionierung
- tanna_op7_refined.png      : Endpunkt der biologischen Entwicklung – waldgrün, siebenstufig

Technische Eigenschaften:
-------------------------
- Format: PNG (transparent)
- Farben: Stufenweise Farbverlauf von #0a2a5e (dunkelblau) bis #228B22 (waldgrün)
- Auflösung: Proportional skaliert auf Basislänge a = 300px
- Hintergrund: transparent
- Visualisierung: Geometrisch korrekt konstruiert, zentriert und kontrolliert
